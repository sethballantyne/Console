InputBuffer

Namespace

console

Definition

			
struct InputBuffer
{
	std::string buffer;
	int x;
	int y;
	int maxBufferLength;
};
buffer Holds all user input as it's typed.
x The X axis coordinate on the console surface where text will be rendered.
y Same as x, but on the Y axis. At the time of writing, this is calculated as being the bottom of the console surface, minus the height of the font.
maxBufferLength The maximum number of characters that can fit into the buffer. At the time of writing, this is simply based on the width of the consoles surface, divided by the fonts character width.

Description

InputBuffer is used by Console to manage user input and where it's rendered; it's documented only for completeness as you generally wont need to interact with this structure. With the exception of buffer, all fields are set when Console_Init is called. Manually setting maxBufferLength to a higher value will result in disappointment, as the buffer doesn't scroll. The general rule of thumb is don't mess with anything in this structure.
