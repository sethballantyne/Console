1. Introduction

Console is a quake-style drop down console written in C++ 11, using SDL 1.2. Given the nature of the project, it's assumed users will want/need to customize Console. For this reason, there's no pre-built libraries, but instead 3 source files that can be added to an existing code base. Console has been designed so it's relatively fail safe: it contains a built in font and uses solid colours to render the console's background. A tool exists to generate fonts from bitmap files into the required 3 dimensional array if you wish to customize the font; see fontgen for details. Section 1.2 also describes how fonts are formatted.

Console is written by Seth Ballantyne and released under the MIT license. The latest version can be found on the projects Github repository.
