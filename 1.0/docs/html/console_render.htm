Console_Render

Summary

#include "console.h"

void console::Console_Render(console::Console& console, SDL_Surface *screen)

Description

Blits the console surface to coordinates 0,0 on the specified surface in screen. As noted in Console_Init, the dimensions of the console are set during initialization to be the same width as the screen parameter, and half its height.

Return Value

None but there should be!
