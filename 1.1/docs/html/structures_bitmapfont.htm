BitmapFont

Namespace

console

Definition

			
struct BitmapFont
{
	SDL_Surface *fontSurface = nullptr;
	Uint32 fontColour;
	Uint32 transparencyColour;
	int characterHeight;
	int characterWidth;
	int numberOfChars;
	char startingChar;
	
	~BitmapFont()
	{
		if(nullptr != fontSurface)
		{
			SDL_FreeSurface(fontSurface);
		}
	}
};
fontSurface The surface containing the font itself.
fontColour The colour of each character in the font. It's converted to a Uint32 via a call to SDL_MapRGB during intialization.
transparencyColour The colour key being used by your sprites for transparency. Like fontColour, this is set during Intialization via a call to SDL_MapRGB.
characterHeight The height of each character in pixels.
characterWidth How wide each character is in pixels.
numberOfChars The number of characters in the font.
startingChar The ASCII value of the first character in the font.

Description

BitmapFont is used to store the result of converting a built-in font to a format that's usable by SDL. The resulting font stored in fontSurface is formatted in such a way that each character is rendered on a single row with no gaps between each character. Generally speaking, you won't need to interact with this structure directly, so it's only documented for completeness.
