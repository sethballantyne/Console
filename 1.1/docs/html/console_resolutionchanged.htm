Console_ResolutionChanged

Summary

#include "console.h"

int console::Console_ResolutionChanged(Console& console, SDL_Surface* screen)

Description

Console_ResolutionChanged reconfigures the console based on the width and height of the surface passed into screen. As the name suggests, this function needs to be called after a resolution change has occurred; failing to do so will result in the console not operating correctly. Note that this function does not resize images used as console backgrounds; if a custom background is being used, you may need to load in a new image to reflect the change in size by calling Console_SetBackground after Console_ResolutionChanged has executed. The same applies to extenal bitmap fonts loaded in via Console_SetFont.

Return Value

CONSOLE_RET_SUCCESS The console was successfully reconfigured.
CONSOLE_RET_CREATE_SURFACE_FAIL A call to SDL_CreateSurface failed while attempting to resize the console.
