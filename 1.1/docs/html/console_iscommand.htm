Console_IsCommand

Summary

#include "console.h"

bool console::Console_IsCommand(console::Console& console, std::string command);

Description

Performs a check to see if command is an existing console command. Note that console commands are not case sensitve.

Return Value

returns true if command is an existing command, otherwise returns false.
