Console_ExecuteCommand

Summary

#include "console.h"

int console::Console_ExecuteCommand(console::Console& console, std::string command, std::vector<std::string>& args);

Description

Executes the console command specified in command with the specified arguments in args. This function is mainly for internal use by the console; Command_ProcessInput uses it to execute commands when the user hits the enter key. If the specified command can't be found, an error message is printed to the console and CONSOLE_RET_COMMAND_DOESNT_EXIST is returned.

There's currently no way to determine whether commands successfully execute, as commands don't have a return value; failure or success as has to be managed within the command's implementation. Note that any uppercase characters in command will be converted to lowercase.

Return Value

CONSOLE_RET_SUCCESS the function executed the specified command.
CONSOLE_RET_NULLPTR_ARGUMENT the function pointer that points to the function implementing the command evaluates to nullptr; this is probably because a nullptr or garbage value was passed to the command_func_ptr parameter in Console_RegisterCommand.
CONSOLE_RET_COMMAND_DOESNT_EXIST the specified command doesn't exist; register it with Console_RegisterCommand.
