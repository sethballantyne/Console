OutputBuffer

Namespace

console

Definition

			
struct OutputBuffer
{
	std::vector buffer;
	int x;
	int y;
	int bottomLineIndex = 0;
	int topLineIndex = 0;
	int maxNumLinesOnScreen; 
	int maxLineLength;
};
buffer Holds all the consoles output. This isn't set to a specific size, so it'll go on forever given the chance.
x The X axis coordinate on the console surface where the output will text start rendering.
y Same as x, but on the Y axis. All output text is rendered on the surface from the bottom up.
bottomLineIndex The index in buffer of the bottommost visible line.
topLineIndex The index in buffer of the topmost visible line. Even if the line has partially scrolled off the screen, it's still considered visible.
maxNumLinesOnScreen The maximum number of lines that can be visible to the user. This is based on the size of the console and the font's height.
maxLineLength The maximum number of characters that can fit into one line.

Description

OutputBuffer is the counterpart to InputBuffer, and as the name suggests, it's used to manage all the console's output. OutputBuffer has only been documented for completeness, as you won't need to directly interact with this structure when incorporating the console to into a project. It's generally not a good idea to directly modify any of the structures members; leave them to the Console_* functions.
