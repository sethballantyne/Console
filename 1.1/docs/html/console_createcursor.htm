Console_CreateCursor

Summary

#include "console.h"

int console::Console_CreateCursor(Console& console, SDL_Colour* colour)

Description

Creates a cursor for the input buffer.The cursor will be the same size as the current font and filled with colour. If colour evaluates to nullptr, the function falls back to the default font colour specified in defaults.h. Generally, you won't need to call Console_CreateCursor, as the console generates a default cursor during initialization for use with the built-in font, and calls to Console_SetFont will result in a cursor being generated for use with the new font.

Return Value

CONSOLE_RET_SUCCESS A new cursor was succesfully created.
CONSOLE_RET_FILL_RECT_FAILED A call to SDL_FillRect failed. While this value technically denotes an error, it's not meant to confer a catastrophic error; at most, the cursor may render incorrectly (if at all), but the console can continue to function.
