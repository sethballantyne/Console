console

Definition

			
struct Console
{
	std::map<std::string, command_func_ptr> commands;
	BitmapFont defaultBitmapFont;
	SDL_Surface *consoleSurface = nullptr;
	InputBuffer inputBuffer;
	OutputBuffer outputBuffer;
	Uint32 defaultConsoleColour;
	bool enabled;

	~Console()
	{
		if(nullptr != consoleSurface)
		{
			SDL_FreeSurface(consoleSurface);
		}
	}
};
commands Holds all commands registered via Console_RegisterCommand.
consoleSurface The surface the font is blitted to when the text is entered into the console. This is blitted to another surface (usually the screen), when Console_Render is called.
inputBuffer Properties related to the space on the console where input is entered. See InputBuffer for details.
outputBuffer Same as inputBuffer, but related to the space where all output is written. See OutputBuffer for details.
defaultConsoleColour The background colour that the console will be filled with. This value is generated by a call to SDL_MapRGB during intitialization.
enabled This is an optional variable that can be used to determine whether the console is currently visible or not. The console itself doesn't make use of it; it exists purely for your convenience.

Description

As the name suggests, Console is the structure that reprensents the console itself and it's generally the only structure you'll need to work with when using the console. All members except for commands and enabled are set when Console_Init is called. Avoid manually modifying these if you don't want to be disappointed.
