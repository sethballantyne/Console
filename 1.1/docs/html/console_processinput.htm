Console_ProcessInput

Summary

#include "console.h"

void console::Console_ProcessInput(console::Console& console, Uint16 unicode);

Description

Console_ProcessInput handles all user input when the console is visible; without calling this, the user will type and not see anything being printed in the input buffer. unicode should be taken from SDL_keysym during a keyboard event in your event handler. As the name suggests, this is a unicode value, so unicode must be enabled by calling SDL_EnableUNICODE(1) during your games initialization. Failing to do this will result in keypresses not being detected.

Return Value

Console_ProcessInput doesn't return any values.
