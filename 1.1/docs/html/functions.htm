This section details the functions used to control the console. With the exception of the Console_*Command functions, you must call Console_Init before using the other functions. The InputBuffer, OutputBuffer and BitmapFont data structures described in the previous section have their own associated functions, but these are intended for internal use only.

The API doesn't contain any clean up functions, as the Console structure contains a destructor that does clean up when it's time. Note that surfaces passed to the console via Console_SetFont and Console_SetBackground are never freed by the console; it's up to the programmer to do this.
