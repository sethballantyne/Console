Console_Print

Summary

#include "console.h"

void console::Console_Print(console::Console& console, std::string message);

Description

Prints message to the console. If the message is too big to fit on a single line, it'll be split into multiple lines. The console will only attempt to print the characters with the range of BitmapFont::firstChar to BitmapFont::lastChar, where BitmapFont is the font loaded in via Console_SetFont. If no font has been loaded in, the range is ASCII 32 (space) to 126 (tilde).

Return Value

Console_Print doesn't return a value.
