Console_SetBackground

Summary

#include "console.h"

void console::Console_SetBackground(Console& console, SDL_Surface* imageSurface)

Description

Sets the console background to the image specified in imageSurface. Passing nullptr will effectively clear the image, forcing the console to use a solid colour background. The colour of the background will be the colour specified during initilization; see Console_Init for more information.

At no point does the console free the surface passed via imageSurface; this is up to the programmer. It's assumed the image being used is the same size as the console. Note that alpha blending is disabled for the surface when Console_SetBackground is called.

Return Value

Console_SetBackground doesn't return a value.
