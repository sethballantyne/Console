Console_Render

Summary

#include "console.h"

void console::Console_Render(console::Console& console, SDL_Surface *screen)

Description

Blits the console surface to coordinates 0,0 on the specified surface in screen. As noted in Console_Init, the dimensions of the console are set during initialization to be the same width as the screen parameter, and half its height.

Return Value

CONSOLE_RET_SUCCESS the function executed successfully without error.
CONSOLE_RET_FILL_RECT_FAILED a call to SDL_FillRect failed.
CONSOLE_RET_BLIT_FAILED a call to SDL_BlitSurface failed.

To get extended error information when Console_Render returns CONSOLE_RET_FILL_RECT_FAILED or CONSOLE_RET_BLIT_FAILED, call SDL_GetError. See the SDL documentation for more information.
