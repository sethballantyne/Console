Console_SetFont

Summary

#include "console.h"

int console::Console_SetFont(Console& console, SDL_Surface* fontSurface, unsigned int numChars, unsigned int charWidth, unsigned int charHeight, unsigned int startingChar)

Description

Sets the console font to the bitmap font specified in fontSurface. If fontSurface evaluates to nullptr, the console will fallback to the built-in font; this also means that nullptr can be used to clear the current font if you wish. In this case, it's safe to pass 0 for the arguments numChars, charWidth, charWidth, and startingChar.

Note that the surface supplied via fontSurface must be using the same colour key supplied during initialization. See Console_Init.

Return Value

CONSOLE_RET_SUCCESS The console's font was successfully changed to the specified font in fontSurface.
CONSOLE_RET_NULLPTR_ARGUMENT fontSurface evaluated to nullptr, resulting in the console falling back to the built-in font.
